> 此规则最初作为“建议”启用，以便内容团队有时间衡量影响并制定清理其存储库的计划。 与阻止 Docs 生成的“错误”和可能影响工作流自动化的“警告”不同，“建议”不需要立即行动，但你应该尽快修复它们，因为在某些时候它们将被升级为“警告”或“错误”。 有关何时更改严重性以及在可用时要包含在消息中的内容的详细信息。
